## 解决痛点

在多语言项目中,前端开发无法专注于业务代码的实现, 经常需要修改多语言文案, 为了解决这一现象本人提出由翻译直接进行对多语言配置, 为了让翻译更好上手, 以及配置的灵活性特此创建本项目
